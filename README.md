## PENJELASAN LANJUTAN

# Nomor 2
(Akses & AksesBaru)
Akses Modifier
- Default
  Variable atau Method dapat diakses oleh class yang sama

- Protected
  Variable atau Method dapat diakses oleh class yang sama
  dan class turunannya dalam satu package

- Private
  Variable atau Method hanya dapat diakses oleh class itu sendiri

- Public
  Variable atau Method dapat diakses oleh class yang sama,
  dalam beda package.
